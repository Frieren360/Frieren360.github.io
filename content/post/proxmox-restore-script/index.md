I created a script to automate the restoration of Proxmox containers and virtual machines to a given snapshot. The script is available on [GitHub](https://github.com/Frieren360/proxmox-restore-script).

It supports restoring from remote nodes using SSH and can restore both LXC containers and QEMU virtual machines.
